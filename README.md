# サクッと!! レストラン検索
- 使用言語：Kotlin 1.4.30
- フレームワーク：Android Studio 4.1.2
- アーキテクチャパターン：MVVM（Model View ViewModel）
- ライブラリ・API：DataBinding・LiveData・ViewModel・Coroutine・Retrofit2・Gson・RecyclerView・Glide

## 概要
予算と検索範囲を入力して、周辺のレストランを検索するアプリです。
